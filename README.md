# Hybrid Modeling for Robust Property Prediction in Circular Al–Mg–Si Extrusions
Notebooks and data from the thesis "Hybrid Modeling for Robust Property Prediction in Circular Al–Mg–Si Extrusions" by Christian Dalheim Øien, submitted for the PhD degree in April 2026.

The notebooks present the methods used in Papers I-IV. Proprietary datasets are replaced by synthetic datasets with the same structure.

### Work in progress
The code and synthetic data in this notebook are now complete, expect for hyperparameter tuning aspects. An update to accomodate this is planned for 25 April 2026.
