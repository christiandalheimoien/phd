# Hybrid Modeling for Robust Property Prediction in Circular Al–Mg–Si Extrusions
Notebooks and data from the thesis "Hybrid Modeling for Robust Property Prediction in Circular Al–Mg–Si Extrusions" by Christian Dalheim Øien, submitted for the PhD degree in April 2026.

The notebooks present the methods used in Papers I-IV. Proprietary datasets are replaced by synthetic datasets with the same structure, generated within the repo for transparency. The focus is methodology reproduction, but unfortunately not results reproduction due to the synthetic stand-ins DS1 for the original D1 and DS2 for the original D2.

Install dependencies:

```bash
pip install -r requirements.txt
```
